Bueno pillate, lo primero es tener en cuenta que vas a tener 4 consolas abiertas, asi es 4 CONSOLAS ABIERTAS, vamos a describir brevemente que hay en cada una

### CONSOLA DE strudelp5.js

Esta consola es la encargada de abrir el bridge para que se envien los datos a las visuales. Si es la primera vez que la abres deberas instalar ciertas dependencias, pilla copia esto paso a paso

npm install -g pnpm@latest-10
pnpm i
pnpm dev

Tene en cuenta que tambien vas a instalar el oscbridge entonces vas a poner tambien
npm install

y despues lo ejecutamos

### CONSOLA DE NPM RUN DEV

Este es mas facil, solo es el encargado de abrir el strudel, entonces solo tenes que instalar:

npm install

y despues npm run dev

### CONSOLA OPEN STAGE CONTROL

Este es le siguiente bridge, que comunica a los controladores del open stage control con las visuales, instalas las dependencias que ya sabes y lo ejecutas
